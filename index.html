Repositorio de la api aqui

La api la deploye a heroku, para dar mas facilidad y no tengas que configurar nada mas, ahi esta el codigo fuente y todas sus funcionalidades.

METHOD URI NAME DESCRIPTION
GET api/companies companies.index Devuelve todas las empresas existentes
POST api/companies companies.store Crea una empresa
PATCH api/companies/{company} companies.show Devuelve una empresa dado el id
DELETE api/companies/{company} companies.update Actualiza una empresa dado el id
GET api/companies/{company} companies.destroy Elimina una empresa dado el id
GET api/companies/{company}/employees companies.employees Devuelve los empleados de una empresa dado el id
GET api/employees employees.index Devuelve todos los empleados existentes
POST api/companies employees.store Crea un empleado
GET api/employees/{employee} employees.show Devuelve un empleado dado el id
PATCH api/employees/{employee} employees.update Actualiza un empleado dado el id
DELETE aapi/employees/{employee} employees.destroy Elimina un empleado dado el id

Otros metodos que se pueden realizar por la url:

  • api/companies?page=2 ---> Devuelve las empresas de la pagina 2
  • api/companies?order=name_company-asc ---> devuelve las empresas ordenadas alfabeticamente en acendente
  • api/companies?order=email-desc ---> Devuelve las empresas ordenadas alfabeticamente con su correo
  • api/employees?order=first_name-asc&page=2 ---> Devuelve los empleads ordenados alfabeticamente por su nombre y la pagina 2
  • api/employees?order=salary-desc&page=5&limit=10 ---> Devuelve los empleados ordenados por su salario en descendencia, pagina 5 y limitando la pagina a 10 empleados
